# NettySpringWebServer
## 预期目标
服务器主要包括四个方面的功能

1.从设备端接收原始数据并进行解析

2.将解析后的数据转发给PC上位机

3.将解析后的数据存储在服务器的MongoDB数据库中

4.通过web查看数据的情况
